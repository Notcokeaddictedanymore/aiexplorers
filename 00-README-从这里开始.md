## 🛠️ 如何把 PRD 转成 Word 文档 / Convert PRD to Word

如果你需要给老师/家长一个 .docx 文件：

### 方法 A：用 Microsoft Word（最简单）
1. 打开 Word
2. 文件 → 打开 → 选 `PRD-AI互动学习课程-完整产品规划.md`
3. Word 会自动识别 Markdown 格式
4. 文件 → 另存为 → 选 `.docx`

### 方法 B：用 macOS 自带 Pages
1. 打开 Pages
2. 拖入 `.md` 文件
3. 另存为 Word 格式

### 方法 C：用在线工具（推荐）
- 上 [pandoc.org/try](https://pandoc.try.fileformat.info/) 或类似的在线 Markdown→Word 转换器
- 注意：不要上传含敏感信息的文档（本 PRD 没有）

### 方法 D：找你的家长帮忙装 Pandoc
```bash
# macOS
brew install pandoc

# 然后
pandoc "PRD-AI互动学习课程-完整产品规划.md" -o "PRD.docx"
```
