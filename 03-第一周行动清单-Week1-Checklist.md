# 第一周行动清单 / Week-One Checklist

> 给 12 岁的你：把这张纸打印出来，贴在书桌前。每完成一项就打勾。
> Print this. Stick it above your desk. Tick the boxes.

---

## 周一：让 PRD 看得见 / Make the PRD visible

- [ ] 把 `PRD-AI互动学习课程-完整产品规划.md` 打印或导出 PDF
- [ ] 在浏览器打开 `01-课程地图-Course-Map.html`，截图 1 张
- [ ] 给爸爸/妈妈讲 5 分钟：你打算做什么，为什么这事重要
- [ ] 听他们提 1 条意见，写在记事本上

🎯 **完成证据**：截图 + 1 条家长反馈

---

## 周二：找到你的"教育顾问" / Find your teacher advisor

- [ ] 列出 3 位可能愿意帮你的老师（信息科技、综合实践、班主任）
- [ ] 给他们写一封简短的邮件/微信（参考模板见下）
- [ ] 先发给最容易答应的那位
- [ ] 附上 PRD 第 5 节（课程架构）和第 7 节（任务样板）

📩 **联系老师模板**：
```
[老师] 您好！

我是 XX 班的 XX。我在做一个面向中小学生的 AI 素养学习课程，灵感来自
Google 的 AI Quests。我已经完成了一份完整的产品规划文档，覆盖 6 章 24 个
互动任务，对齐了教育部 2025 通识指南。

希望您能花 15 分钟看一眼第 5 节（课程地图）和第 7 节（一个完整任务样板），
帮我提一条改进意见。任何反馈都非常宝贵——尤其是哪里"不切实际"。

附件是 PDF（不到 30 页），最近的截图链接是 ……

谢谢老师！
XX
```

🎯 **完成证据**：3 位老师中至少 1 位答应阅读

---

## 周三：建项目大本营 / Set up your HQ

- [ ] 在 Notion / 印象笔记 / 一个纸笔本上建项目仓库
- [ ] 第一页：今天写 5 行（做了什么、卡在哪、明天计划）
- [ ] 第二页：把"24 个任务清单"列出来（见 PRD 第 5 节）
- [ ] 第三页：建一个"灵感库"，每天看到 AI 相关新闻就记下来

🎯 **完成证据**：项目本子里有 3 页

---

## 周四：第一个 GitHub 账号 / Your first GitHub

- [ ] 去 github.com 注册账号（用 jiang.gu@icloud.com 即可）
- [ ] 新建一个仓库：`ai-quests-china`（公开就好）
- [ ] 把 PRD 的 Markdown 上传上去（点上传按钮就行，不用命令行）
- [ ] 打开 Settings → Pages → 启用 GitHub Pages
- [ ] 你的网址自动生成：`https://[你的用户名].github.io/ai-quests-china/`
- [ ] 把这个网址发给你的"教育顾问"老师

📌 **如果 GitHub 打不开**：用 Gitee.com（国内可访问）作为替代

🎯 **完成证据**：一个可以访问的网址

---

## 周五：决定要做的第一章 / Pick your first chapter

- [ ] 翻 PRD 第 5 节，把 6 章选项重新读一遍
- [ ] 推荐先做 **Ch.1（AI 怎么看见世界 · 大熊猫识别）**，理由：
   - 视觉效果好，照片漂亮，容易拍出 demo
   - 概念基础，可以同时教"训练数据""标签""分类"三个核心词
   - 有 Teachable Machine 可以直接嵌入，零代码
   - 中国大熊猫文化背景熟悉，老师好讲

- [ ] 写下你为什么选这一章（≥ 50 字）
- [ ] 把 PRD 7.1 节（M1-3 熊猫任务样板）再读一遍，做笔记

🎯 **完成证据**：一段自己写的"我为什么先做这章"

---

## 周六：开始美术资源 / Start visual assets

- [ ] 注册 Canva 免费账号（canva.cn 即可）
- [ ] 看一个 Canva 入门视频（B 站 10 分钟）
- [ ] 设计你的"智小灵"形象：用 Canva 的图形 + 简笔画
- [ ] 找 12 张大熊猫照片（百度图片，注意标"署名-相同方式共享 CC"协议）
- [ ] 找 6 张小熊猫/浣熊/棕熊照片
- [ ] 全部命名为 `panda-01.jpg`、`notpanda-01.jpg` 这样

🎯 **完成证据**：一个文件夹里有 18 张图 + 1 个智小灵设计稿

---

## 周日：让 AI 开始为你写代码 / Make AI work for you

- [ ] 打开 Claude（你现在用的这个）或 ChatGPT
- [ ] 把 PRD 第 7.1 节（M1-3 任务样板）复制粘贴
- [ ] 说："请按这个样板，给我写一个完整的 HTML 文件，单页面，
       不需要任何后端，使用 Tailwind CSS（CDN 引入），
       把照片用占位图代替，做出来一个能在浏览器打开的可玩任务。"
- [ ] AI 会给你代码，复制保存为 `m1-3-panda.html`
- [ ] 双击打开，看看效果
- [ ] 不满意就让 AI 改——"按钮太小"、"颜色太暗"、"加一个智小灵对话框"

🎯 **完成证据**：电脑桌面上有一个 m1-3-panda.html，能在浏览器打开

---
